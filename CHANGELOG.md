## [0.1.0] - 2025-10-18

### 新增

-   初始版本发布
-   基于大漠插件的 Python 自动化封装库
-   核心功能模块：
    -   鼠标操作模块 (`mouse.py`)
    -   键盘操作模块 (`key.py`)
    -   图像识别模块 (`apiproxy.py`)
    -   核心引擎模块 (`coreengine.py`)
    -   窗口枚举模块 (`enum_wind.py`)
-   配置管理系统：
    -   基础配置管理 (`config.py`)
    -   安全认证信息管理 (`secure_config.py`)
-   依赖检查工具 (`dependencies.py`)
-   时间工具模块 (`time_utils.py`)
-   大漠插件注册管理 (`regsvr.py`)
-   主执行类 (`damo.py`)

## 版本说明

### 版本号格式

-   **主版本号**：不兼容的 API 修改
-   **次版本号**：向下兼容的功能性新增
-   **修订号**：向下兼容的问题修正

### 变更类型

-   **新增**：新功能
-   **修复**：问题修复
-   **变更**：现有功能的变更
-   **移除**：功能的移除
-   **安全**：安全相关的修复
